# Architecture Review Checklist

| Item | Review Focus | Severity Guidance | Suggested Fix / Notes |
| --- | --- | --- | --- |
| A1 | Confirm space planning aligns with client workplace standards (desk density, collaboration ratios) | Major if deviation >10% | Rebalance program, adjust workstation counts per client standard. |
| A2 | Verify fire egress paths meet SBC width and travel distance limits | Critical if non-compliant | Increase corridor width, re-route paths, add exit signage. |
| A3 | Ensure door swings and clearances satisfy accessibility (MOMRAH, SBC) | Major if partial obstruction | Adjust door locations/swing direction, revise hardware. |
| A4 | Check ceiling heights vs client guidelines and SBC minimums | Minor unless impacting MEP routing | Coordinate with MEP, adjust bulkhead heights. |
| A5 | Confirm finish selections match client brand palette and durability requirements | Minor unless missing specs | Update finish schedule, attach manufacturer data. |
| A6 | Validate acoustic treatments in focus rooms per best practice | Major if below NRC 0.7 | Increase acoustic panels, add seals. |
| A7 | Review coordination of architectural openings with MEP devices | Major if clashes | Shift diffusers, relocate lights, adjust soffits. |
| A8 | Verify compliant signage strategy (wayfinding, bilingual) | Minor | Update signage package, confirm Arabic/English text. |
