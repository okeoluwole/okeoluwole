# ICT & Low Voltage Checklist

| Item | Review Focus | Severity Guidance | Suggested Fix / Notes |
| --- | --- | --- | --- |
| I1 | Structured cabling pathways sized for growth | Major if <25% spare | Upsize trays/conduits, coordinate with ceiling. |
| I2 | Server/data room layout meets client Tier requirements | Critical if missing redundancy | Add A/B power, hot/cold aisle containment. |
| I3 | Wi-Fi access point density per workplace best practice | Major if coverage gaps | Increase APs, coordinate with ceiling plan. |
| I4 | Security systems (CCTV, ACS) comply with client and Saudi regulations | Major if blind spots | Add cameras/readers, ensure data retention compliance. |
| I5 | Integration with BMS and workplace apps | Minor unless critical function missing | Update interface specs, confirm APIs. |
| I6 | Pathways separated from high-voltage per SEC | Critical if shared | Provide dedicated containment, maintain clearances. |
