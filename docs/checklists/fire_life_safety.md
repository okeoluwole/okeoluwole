# Fire & Life Safety Checklist

| Item | Review Focus | Severity Guidance | Suggested Fix / Notes |
| --- | --- | --- | --- |
| F1 | Fire compartmentation and wall ratings per SBC 707 & Civil Defense | Critical if breach | Reinstate rated walls/doors, specify fire stopping. |
| F2 | Exit signage, emergency lighting, and photometric coverage | Major if partial | Add fixtures/signage per NFPA 101 equivalents. |
| F3 | Sprinkler coverage coordinated with architectural elements | Critical if shadowed | Relocate heads, add sidewall heads. |
| F4 | Fire alarm device layout and zoning | Critical if missing devices | Add detectors, synchronize with existing panel zoning. |
| F5 | Fire-fighting equipment (hose reels, extinguishers) placement | Major if spacing >30m | Adjust locations, add cabinets. |
| F6 | Smoke control and pressurization strategy | Major if not defined | Provide calculations, confirm fan sizing. |
| F7 | Refuge areas and accessibility in high-rise contexts | Major if absent | Designate refuge rooms, ensure 2-hour rating. |
