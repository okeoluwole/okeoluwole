# Accessibility Checklist

| Item | Review Focus | Severity Guidance | Suggested Fix / Notes |
| --- | --- | --- | --- |
| AC1 | Barrier-free entrance routes per SBC 1101 and MOMRAH | Critical if non-compliant | Provide ramps, adjust slopes, add tactile indicators. |
| AC2 | Door clear opening width ≥ 915 mm and hardware accessible | Major if <915 mm | Widen doors, change hardware to lever type. |
| AC3 | Accessible restrooms with compliant turning radius and grab bars | Critical if missing | Reconfigure layout, install grab bars at 850 mm AFF. |
| AC4 | Vertical circulation – lifts, platform lifts, signage | Major if missing features | Add tactile signage, ensure audible/visual indicators. |
| AC5 | Workstation reach ranges and adjustable furniture | Minor unless non-compliant | Provide height-adjustable desks, accessible storage. |
| AC6 | Assistive technology accommodations (hearing loops, visual alarms) | Minor unless mandated | Coordinate with AV/ICT team, install loops where required. |
